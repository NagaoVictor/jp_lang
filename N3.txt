～ようにしている — Esforçar-se para manter um hábito.
例: 毎日野菜を食べるようにしています。
 

～ことになっている — Está decidido / é regra.
例: 学校では日本語を話すことになっています。
 

～ように言われる — Ser instruído / mandado a fazer algo.
例: 先生に宿題を出すように言われました。
 

～てしまう — Terminar / acabar (às vezes com arrependimento).
例: 財布を忘れてしまいました。
 

～ところだ — Estar prestes a / no meio de / ter acabado de.
例: 今から出かけるところです。
 

～ばかりだ — Continua a / tendência de.
例: 物価が上がるばかりです。
 

～によって／～により — De acordo com / por meio de.
例: 国によって文化が違います。
 

～られる（受身形） — Forma passiva / receber ação.
例: 先生にほめられました。
 

～させる（使役形） — Fazer alguém fazer algo / permitir.
例: 子供に野菜を食べさせました。
 

～ても — Mesmo se / mesmo que.
例: 雨が降っても行きます。
 

～ずに — Sem fazer algo (formal de ～ないで).
例: 朝ごはんを食べずに出かけた。
 

～ようとする — Tentar fazer algo.
例: 出かけようとしたとき、電話が鳴りました。
 

～ようとしない — Não fazer esforço para / não tentar.
例: 彼は勉強しようとしません。
 

～にくい／～にくさ — Difícil de fazer / dificuldade.
例: この文は理解しにくいです。
 

～やすい／～やすさ — Fácil de fazer / facilidade.
例: このアプリは使いやすいです。
 

～たばかり — Acabou de fazer algo.
例: 今来たばかりです。
 

～ば～ほど — Quanto mais..., mais...
例: 勉強すればするほど上手になります。
 

～のように／～のような — Como / semelhante a.
例: 鳥のように飛びたい。／夢のような話です。
 

～ために／～ための — Para / com o objetivo de.
例: 健康のために毎日運動しています。
 

～に対して（は）／～に対する — Em relação a / contra.
例: 社長に対して意見を言いました。
 

～について／～に関して — Sobre / a respeito de.
例: 日本の歴史について調べています。
 

～にとって — Para (ponto de vista pessoal).
例: 私にとって家族は一番大切です。
 

～によると／～によれば — De acordo com (fonte).
例: 天気予報によると、明日は雨です。
 

～ことに — Expressa sentimento forte.
例: 驚いたことに、彼は日本語がペラペラです。
 

～わけではない — Não quer dizer que / não necessariamente.
例: 甘い物が嫌いなわけではありません。
 

～わけがない — Impossível que / não há como.
例: そんなことできるわけがない。
 

～ような気がする — Ter a impressão de que.
例: 今日はいいことがありそうな気がします。
 

～ように見える — Parece / aparenta ser.
例: 彼は元気なように見えます。
 

～ほど～ない — Não tanto quanto.
例: 東京は大阪ほど暑くないです。
