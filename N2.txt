～に違いない — Sem dúvida / certamente.
例: 彼は学生に違いない。
 

～にほかならない — Nada mais do que / precisamente.
例: 成功の理由は努力にほかならない。
 

～限り／～限りでは — Enquanto / até onde.
例: 私の知る限りでは、彼は優しい人です。
 

～にせよ／～にしろ — Mesmo que / ainda que.
例: 行くにせよ行かないにせよ、早く決めてください。
 

～にしても — Mesmo considerando que.
例: 高いにしても、質がいいです。
 

～ことはない — Não há necessidade de.
例: 心配することはない。
 

～わけがない — Impossível que.
例: そんなことできるわけがない。
 

～ものだ — Expressa lembrança, costume ou conselho.
例: 若いころはよく走ったものだ。
 

～ものだから — Porque / motivo pessoal.
例: 遅れたのは渋滞だったものだから。
 

～ものの — Embora / apesar de.
例: 知っているものの、説明できない。
 

～ものか／～もんか — De jeito nenhum / nunca.
例: もう二度と行くもんか。
 

～ものではない — Não se deve / é inapropriado.
例: 約束を破るものではない。
 

～ないものでもない — Talvez seja possível.
例: 頑張ればできないものでもない。
 

～に越したことはない — É melhor que / ideal.
例: 安いに越したことはない。
 

～とは思えない — Não dá para acreditar que.
例: 彼がそんなことをするとは思えない。
 

～からすると／～からすれば — A julgar por / do ponto de vista de.
例: 顔からすると、彼は疲れているようだ。
 

～にしては — Para / considerando que.
例: 子どもにしてはよくできた。
 

～わりに — Embora / apesar de.
例: 値段のわりに美味しい。
 

～っぽい — Parece / jeito de.
例: 子どもっぽい話し方。
 

～がち — Tende a / propenso a.
例: 冬は風邪をひきがちです。
 

～気味 — Um pouco / levemente.
例: 風邪気味です。
 

～っぱなし — Deixar algo como está.
例: ドアを開けっぱなしにしないで。
 

～きり — Desde / apenas / depois de.
例: 彼とは去年会ったきりです。
 

～だらけ — Cheio de / coberto de.
例: ほこりだらけの部屋。
 

～向け — Voltado para / destinado a.
例: 子ども向けの映画です。
 

～向き — Apropriado para.
例: この服は冬向きです。
 

～通り／～どおり — Como / conforme.
例: 言った通りにやってください。
 

～っけ — Mesmo? / era isso? (lembrança informal).
例: 明日だったっけ？
 

～なんか／～なんて — Algo como / surpresa / desprezo.
例: 私なんかまだまだです。
 

～とか — Coisas como / e tal.
例: 寿司とかラーメンが好きです。
 

～だって — Até mesmo / ouvi dizer.
例: 子どもだってわかるよ。
