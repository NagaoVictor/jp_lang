～ようになる — Passar a ser / tornar-se possível.
例: 日本語が話せるようになりました。
 

～ておく — Fazer algo antecipadamente.
例: 旅行の前にチケットを買っておきます。
 

～ば／～たら — Condicional “se”.
例: 雨が降ったら、出かけません。
 

～と — Sempre que / ao fazer algo.
例: 春になると花が咲きます。
 

～までに — Até (prazo).
例: 明日までに宿題を出してください。
 

～ながらも — Apesar de / mesmo fazendo.
例: 忙しいながらも楽しいです。
 

～ほど — A ponto de / comparativo.
例: 死ぬほど疲れた。
 

～ために — Para / com o propósito de.
例: 日本語を勉強するために日本へ来ました。
 

～ように言う — Dizer para que alguém faça algo.
例: 先生は学生にもっと勉強するように言いました。
 

～てはいけない — Proibição.
例: ここでタバコを吸ってはいけません。
 

～てもいい — Permissão.
例: この部屋に入ってもいいです。
 

～たことがある — Ter feito algo / experiência.
例: 富士山に登ったことがあります。
 

～や～など — Exemplos / “como”, “etc.”
例: 寿司や天ぷらなどが好きです。
 

～しか～ない — Apenas / nada além de.
例: 100円しか持っていません。
 

～について — Sobre / a respeito de.
例: 日本の文化について勉強しています。
 

～とき — Quando / momento.
例: 子供のとき、アニメが好きでした。
 

～まえに／～あとで — Antes de / depois de.
例: 寝るまえに歯を磨きます。／勉強のあとでゲームをします。
 

～あいだ（に） — Durante / enquanto.
例: 授業のあいだ、静かにしてください。
 

～ようだ — Parece que / aparência.
例: 雨が降りそうなようだ。
 

～みたい — Parece / estilo informal de ようだ.
例: あの人は先生みたいです。
 

～らしい — Ouvi dizer que / aparentar ser.
例: 田中さんは結婚したらしい。
 

～のに — Apesar de / embora.
例: 雨なのに出かけました。
 

～くなる／～になる — Tornar-se (mudança de estado).
例: 暖かくなりました。／先生になりました。
 

～やすい／～にくい — Fácil / difícil de fazer (revisão com adjetivo).
例: この靴は歩きやすいです。
 

～はじめる／～おわる — Começar / terminar de fazer algo.
例: 勉強をはじめました。／仕事がおわりました。
 

～ているところ — Estar no meio de fazer algo.
例: 今、昼ごはんを食べているところです。
 

～ばかり — Acabou de / só faz algo.
例: 食べたばかりです。／寝てばかりいます。
 

～ように見える — Parece que (impressão visual).
例: 元気なように見えます。
 

～ことができる — Poder / ser capaz de fazer algo.
例: 漢字を読むことができます。
 

～にする — Escolher / decidir.
例: コーヒーにします。
