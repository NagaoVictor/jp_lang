～わけではないが — Não exatamente, mas...
例: 嫌いなわけではないが、あまり食べない。
 

～ないことはない — Não é impossível / até que dá pra.
例: 行けないことはないけど、ちょっと遠い。
 

～なくはない — Não é impossível / talvez sim.
例: 分からなくはない。
 

～ないものでもない — Pode até ser possível.
例: 頑張ればできないものでもない。
 

～ことはない — Não há necessidade de.
例: 心配することはないよ。
 

～というわけではない — Não quer dizer que.
例: 全部好きというわけではない。
 

～とはいえ — Embora / apesar de.
例: 冬とはいえ、今日は暖かい。
 

～にせよ／～にしろ — Mesmo que / ainda que.
例: 行くにせよ行かないにせよ、早く決めてください。
 

～にほかならない — Nada mais do que / precisamente.
例: 成功の理由は努力にほかならない。
 

～ない限り — A menos que / enquanto não.
例: 雨が止まない限り、試合はできない。
 

～限りでは — Até onde / dentro do limite.
例: 私が知る限りでは、彼は優しい人です。
 

～に限って — Justamente / só nesse caso.
例: 忙しい日に限って雨が降る。
 

～かと思うと／～かと思ったら — Logo depois / quase ao mesmo tempo.
例: 雨が降ったかと思うと、すぐに止んだ。
 

～か～ないかのうちに — Assim que / logo quando.
例: ベルが鳴るか鳴らないかのうちに、学生が出て行った。
 

～次第 — Assim que / dependendo de.
例: 準備ができ次第、出発します。
 

～最中に — No meio de / enquanto.
例: 会議の最中に電話が鳴った。
 

～ついでに — Aproveitando que / enquanto faz algo.
例: 買い物のついでに郵便を出した。
 

～に違いない — Certamente / sem dúvida.
例: 彼が犯人に違いない。
 

～わけがない — Impossível que.
例: そんなのできるわけがない！
 

～ものだ — Expressa lembrança, costume ou conselho.
例: 若いころはよく走ったものだ。
 

～ものだから — Porque / motivo pessoal.
例: 遅れたのは渋滞だったものだから。
 

～ものの — Embora / apesar de.
例: 知っているものの、説明できない。
 

～ものか／～もんか — De jeito nenhum / nunca.
例: もう二度と行くもんか。
 

～ものではない — Não se deve / é inapropriado.
例: 約束を破るものではない。
 

～ないものでもない — Talvez seja possível.
例: 頑張ればできないものでもない。
 

～に越したことはない — É melhor que / ideal.
例: 安いに越したことはない。
 

～とは思えない — Não dá pra acreditar que.
例: 彼がそんなことをするとは思えない。
 

～にすぎない — Nada mais que / apenas.
例: 僕は学生にすぎません。
 

～からすると／～からすれば — A julgar por / do ponto de vista de.
例: 顔からすると、彼は疲れているようだ。
 

～にしては — Para / considerando que.
例: 子どもにしてはよくできた。
